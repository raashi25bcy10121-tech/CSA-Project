# Hospital Appointment Scheduler

A Python-based appointment management system designed to streamline healthcare scheduling operations and eliminate double-booking incidents.

## Overview

This Appointment Scheduler provides a comprehensive solution for healthcare facilities to manage patient appointments efficiently. It addresses critical challenges in manual scheduling systems by preventing overlapping appointments, tracking appointment status, and organizing daily schedules.

## Features

- **Double-Booking Prevention**: Automated conflict detection for doctors' time slots
- **Flexible Scheduling**: Schedule appointments with patient details, doctor assignment, and reason for visit
- **Appointment Status Tracking**: Track scheduled, completed, and cancelled appointments
- **Viewing Options**: Filter appointments by date, doctor, or view all
- **Daily Schedule Generation**: Organized time-sorted view of daily appointments
- **Simple Cancellation & Completion**: Easy appointment status management

## Technologies Used

- **Programming Language**: Python 3
- **Data Structure**: Dictionary-based in-memory storage
- **No External Dependencies**: Pure Python implementation
